## Installation

Just add it to your ``rebar.config`` deps:

```erlang
  {deps, [
    ...
    {kla, ".*",
      {git, "git@github.com:lfex/kla.git", "master"}}
      ]}.
```

And then do the usual:

```bash
    $ rebar get-deps
    $ rebar compile
```
